📊 Blinkit Data Analysis (Power BI Project)



📌 Overview

This project analyzes Blinkit's sales and customer behavior data using Power BI. The goal is to extract meaningful insights about product performance, customer purchasing patterns, discounts, and revenue trends.

🎯 Objectives

Identify top-selling products
Analyze the impact of discounts on purchases
Compare customer segments (subscribed vs non-subscribed)
Track revenue trends and spending behavior
Build interactive dashboards for decision-making

🛠️ Tools & Technologies

Power BI (.pbix file)
Data Cleaning & Transformation (Power Query)
Data Modeling
DAX (Data Analysis Expressions)

📂 Project Structure

Blinkit Data Analysis.pbix   # Main Power BI dashboard file
README.md                   # Project documentation

📊 Key Insights

Products with the highest discount-driven purchases
Customer spending patterns
Revenue comparison between subscriber and non-subscriber users
Seasonal or category-based trends

📈 Dashboard Features

Interactive filters (category, product, customer type)
KPI cards (Total Revenue, Average Spend, etc.)
Visual charts (bar charts, pie charts, line graphs)
Drill-down capabilities for deeper analysis
